# Implementacion_Neuronas_Clasificacion_Binaria-Frutas-
Implementación de clasificador Manzana/Naranja con RNA simple, Arduino Uno y sensor TCS3200.  El sistema adquiere datos de color, los normaliza de forma min-max invertida (periodo a intensidad 0-255), y los usa para entrenar una red 4-4-1 en Python. Los Pesos y Bias se implementan en Arduino para la clasificación autónoma y en tiempo real.
